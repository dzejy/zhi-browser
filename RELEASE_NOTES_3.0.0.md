# Zhi Browser 3.0.0

Zhi Browser 3.0.0 是在 2.0.0 高级功能基础上的大版本增强，重点补齐浏览器本体能力，包括下载、代理、历史、书签、密码、无痕、搜索入口、全屏入口，以及页面按钮自定义。

Zhi Browser 不是普通 WebView 壳，而是一个面向个人长期自用的 Windows 桌面 AI 浏览器。

## 项目定位

Zhi Browser 的目标不是复刻 Chrome，而是做一个更适合个人长期使用的 AI 浏览器。

核心方向：

- 界面干净
- 功能可控
- 深色 UI
- 多标签浏览
- 浏览器数据可被 AI 利用
- 支持资源处理、阅读、脚本、下载、代理等高级能力
- 后续可以继续扩展为个人知识浏览器和 AI 工作台

## 技术栈

- Electron
- React
- TypeScript
- WebContentsView
- electron-vite
- 自定义浏览器 chrome
- Windows 桌面端优先

当前 Windows 版使用 BaseWindow + WebContentsView 路线。旧的 webview 路线已经废弃，不再继续推进。

## 3.0.0 新增功能

### 1. 内置多线程下载器

浏览器内置自己的下载核心，不再完全依赖外部下载器。

主要能力：

- 多线程下载
- 断点续传
- 暂停、恢复、取消
- Range 支持检测
- 单线程自动回退
- 下载任务持久化
- 分片合并
- 与 IDM、FDM、NDM 外部下载器共存

### 2. 内置代理

3.0.0 新增内置代理能力，使用 mihomo 作为代理核心。

当前定位：

- 浏览器内部可控代理
- 不完全依赖系统代理
- 为后续站点规则、标签页规则、代理状态提示打基础

### 3. 独立历史记录管理页

历史记录不再只是简单列表，而是独立管理页面。

支持方向：

- 查看历史记录
- 搜索历史记录
- 删除历史记录
- 快速重新打开历史网页
- 为 AI 搜索历史提供数据基础

### 4. 独立书签管理页

书签管理从简单收藏按钮升级为独立页面。

支持方向：

- 查看全部书签
- 搜索书签
- 打开收藏网页
- 删除和整理书签
- 为 AI 搜索书签提供数据基础

### 5. 简易密码管理器

提供基础账号密码保存和管理能力。

当前定位是简易可用版，不替代专业密码管理器。后续需要继续做安全加固，例如本地加密、主密码保护、敏感数据隔离等。

### 6. 无痕模式

新增隐私浏览模式，用于临时访问网页。

目标：

- 减少普通历史记录污染
- 减少缓存和登录状态留存
- 提供更干净的临时浏览环境

### 7. AI 搜索历史记录和书签

AI 不再只围绕当前网页工作，也可以连接浏览器内部数据。

可以用于：

- 从历史记录里找之前访问过的网页
- 从书签里找相关资料
- 用自然语言搜索个人浏览轨迹
- 把历史和书签变成轻量个人知识库

### 8. 地址栏旁快捷搜索框

在主地址栏旁边增加独立快捷搜索框。

目的：

- 地址栏负责网址
- 搜索框负责快速搜索
- 降低输入混乱
- 提高日常查资料效率

### 9. 左上角全屏模式入口

提供明确的全屏按钮入口。

适合：

- 沉浸式阅读
- 视频网页
- 分屏浏览
- 长文档查看
- 减少浏览器顶部 chrome 干扰

### 10. 页面按钮自定义开关

页面按钮可以按个人习惯启用或隐藏。

解决的问题：

- 功能越来越多后，按钮容易堆满
- 不常用功能可以隐藏
- 常用功能可以保留
- 浏览器界面复杂度可控

## 2.0.0 已有功能继承

3.0.0 继承了 2.0.0 的高级浏览器工具能力，包括：

- 用户脚本引擎，类似 Tampermonkey
- 阅读模式
- 资源嗅探
- 外部下载器对接，支持 IDM、FDM、NDM
- 标签页预览
- 侧边栏网页面板
- 鼠标手势
- 分屏浏览
- 暗色模式
- 沉浸式翻译
- AI 四个网页动作：
  - 省流版本
  - 丁真一下
  - 全网通缉
  - 大司马模式

## AI 功能

Zhi Browser 的 AI 功能围绕网页和浏览器数据展开。

| 功能名 | 作用 |
|---|---|
| 省流版本 | 总结当前网页 |
| 丁真一下 | 验证网页内容是否可靠 |
| 全网通缉 | 围绕当前主题搜索补充信息 |
| 大司马模式 | 站在反方视角批判网页观点 |

3.0.0 后，AI 开始进一步连接历史记录和书签，让浏览器内部数据可以被自然语言搜索和整理。
