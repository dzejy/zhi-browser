# Zhi Browser 2.0.0

Zhi Browser 2.0.0 是当前正式主版本。该版本以美化版作为后续主线，采用深色精修 UI，同时保留传统横向标签页和正常浏览器使用习惯。

本 Release 同时提供两个安装包：

## 下载选择

### Zhi Browser Setup 2.0.0.exe

美化版主版本。  
这是后续继续更新的主线版本，采用深色精修 UI，整体视觉质感更完整。

### Zhi Browser Simple Setup 2.0.0.exe

简洁版最终版本。  
这不是功能阉割版，而是保留简洁 UI 风格的完整功能版本。它同样包含 Zhi Browser 2.0.0 的核心能力和新增功能模块，只是界面路线更轻量、直接、克制。

## 2.0.0 核心功能

- 多标签页浏览
- 地址栏输入与跳转
- 后退 / 前进 / 刷新
- 书签栏
- 历史记录
- 下载管理
- 设置持久化
- 窗口状态记忆
- 侧边面板系统
- 广告拦截
- Electron + WebContentsView 浏览器架构

## AI 网页工作流

- 省流版本：总结当前网页内容
- 丁真一下：验证网页事实，检查可能的错误
- 全网通缉：围绕当前网页主题搜索补充信息
- 大司马模式：站在反方角度反驳网页观点，找出逻辑漏洞

## 2.0.0 新增功能模块

1. 用户脚本引擎
2. 阅读模式
3. 资源嗅探
4. 外部下载器对接
5. 标签页预览
6. 侧边栏网页面板
7. 鼠标手势
8. 分屏浏览

## 其他增强功能

- 暗色模式
- 沉浸式翻译
- AI 设置页面
- 搜索引擎配置
- 外部下载器配置
- 广告拦截白名单
- 用户脚本管理面板
- 资源嗅探面板

## 版本路线说明

从 2.0.0 开始，Zhi Browser 主线将以美化版 main 为主继续开发。

Simple 2.0.0 作为功能完整的简洁版最终版本长期保留。
