# Zhi Browser\n\n**Zhi Browser** 是一个个人 AI 辅助浏览器项目，基于 Electron、React、TypeScript、Electron Vite 和 WebContentsView 开发。\n\n当前版本：**1.0.0S Simple 简洁版**\n\nZhi Browser 的目标是做一个真正适合个人使用的 AI 浏览器壳。它保留传统横向标签页和正常浏览器操作习惯，同时把 AI 总结、验证、搜索、反驳这些网页阅读能力接进浏览器内部。\n\n---\n\n## 下载\n\n请在 GitHub Releases 页面下载最新版 Windows 安装包。\n\n当前推荐下载：\n\n**Zhi Browser Setup 1.0.0S.exe**\n\n这是当前的 1.0.0S Simple 简洁版安装包，适合正常安装使用。\n\n`win-unpacked` 目录主要用于本地测试和开发调试，普通使用只需要下载 `Setup.exe` 安装包。\n\n---\n\n## 当前版本：1.0.0S Simple 简洁版\n\n`1.0.0S` 是 Zhi Browser 第一个正式打包发布的版本。\n\n版本含义：\n\n- **S = Simple / 简洁版**\n- 当前版本作为后续美化前的简洁基线\n- 下一阶段版本是 **1.0.0-B Beautiful 美化版**\n\n当前版本已经完成主要可用基础：\n\n- 多标签页浏览\n- 横向标签栏布局\n- 地址栏导航\n- 浏览器工具栏\n- WebContentsView 页面渲染\n- Electron BaseWindow 架构路线\n- 设置中心\n- 独立 AI 设置页面\n- AI 侧边栏\n- AI 网页功能按钮\n- 历史记录面板\n- 书签面板\n- 下载面板\n- ADB 相关入口\n- 深色界面基础风格\n- 红色 Z 官方图标\n- Windows NSIS 安装包\n\n目前项目整体完成度约 **70%**，已经具备个人日常使用和继续迭代的基础。\n\n---\n\n## 核心功能\n\n### 浏览器基础能力\n\nZhi Browser 当前已经具备一个桌面浏览器的基础结构：\n\n- 多标签页\n- 横向标签栏\n- 地址栏输入与跳转\n- 后退 / 前进 / 刷新\n- 侧边面板\n- 设置持久化基础\n- 深色桌面应用界面\n- Electron + WebContentsView 渲染结构\n\n项目采用 **Electron + BaseWindow + WebContentsView** 路线，不再继续使用早期失败的 `<webview>` 路线。\n\n### AI 网页工作流\n\nZhi Browser 内置 AI 侧边栏，并围绕“当前网页”设计了几个固定 AI 功能。\n\nAI 不是单纯的聊天框，而是直接服务于浏览网页时的阅读、判断和分析。\n\n当前 AI 功能包括：\n\n| 功能名称 | 作用 |\n| --- | --- |\n| 省流版本 | 总结当前网页内容 |\n| 丁真一下 | 验证网页事实，检查可能的错误 |\n| 全网通缉 | 围绕当前网页主题搜索相关信息和补充材料 |\n| 大司马模式 | 站在反方角度反驳网页观点，找逻辑漏洞 |\n\n目标使用流程：\n\n1. 打开一个网页。\n2. 呼出 AI 侧边栏。\n3. 选择总结、验证、搜索或反驳。\n4. 用 AI 帮助理解网页，而不是只被动阅读。\n\n### 设置与面板\n\n当前已经包含：\n\n- 普通设置页面\n- 独立 AI 设置页面\n- AI 侧边栏\n- 历史记录面板\n- 书签面板\n- 下载面板\n- ADB 相关入口\n- 搜索引擎配置基础\n- API / 模型配置基础\n\nAI 设置已经从普通设置中拆分出来，方便后续继续扩展 API Key、模型选择、联网搜索、提示词模板等功能。\n\n---\n\n## 视觉方向\n\nZhi Browser 当前采用深色、紧凑、桌面应用风格的界面。\n\n当前 1.0.0S Simple 简洁版重点是：先把浏览器核心功能和基础 AI 工作流跑通，保持界面简洁、稳定、可用。\n\n下一阶段会进入 1.0.0-B Beautiful 美化版，重点提升整体质感、交互细节和日常使用舒适度。\n\n视觉方向会参考 Arc Browser 和 Zen Browser 的一些质感，但不会改成它们那种竖向 sidebar 浏览器。Zhi Browser 会继续保持自己的横向标签页形态。\n\n---\n\n## 版本规划\n\n### 1.0.0S Simple 简洁版\n\n状态：已完成 / 准备发布到 GitHub Release\n\n定位：后续美化前的简洁基线\n\n安装包名称：\n\n**Zhi Browser Setup
