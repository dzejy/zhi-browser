# Zhi Browser

Zhi Browser 是一个个人 AI 辅助浏览器项目，基于 Electron、React、TypeScript、Electron Vite 和 WebContentsView 开发。

**当前版本：2.0.0**

Zhi Browser 的目标是做一个真正适合个人使用的 AI 浏览器壳。它保留传统横向标签页和正常浏览器操作习惯，同时把 AI 总结、验证、搜索、反驳这些网页阅读能力接进浏览器内部，并集成了用户脚本引擎、暗色模式、沉浸式翻译、阅读模式、资源嗅探、鼠标手势、分屏浏览等高级功能。

## 核心功能

### 浏览器基础能力

- 多标签页 + 横向标签栏
- 地址栏输入与跳转
- 后退 / 前进 / 刷新
- 书签栏（可开关）
- 窗口状态记忆（位置、大小、最大化状态）
- 设置为默认浏览器
- 侧边面板系统
- 设置持久化
- 深色桌面应用界面
- Electron + BaseWindow + WebContentsView 渲染结构

### AI 网页工作流

内置 AI 侧边栏，围绕当前网页设计固定 AI 功能，打开即自动聚焦输入框。

| 功能名称 | 作用 |
|---------|------|
| 省流版本 | 总结当前网页内容 |
| 丁真一下 | 验证网页事实，检查可能的错误 |
| 全网通缉 | 围绕当前网页主题搜索相关信息和补充材料 |
| 大司马模式 | 站在反方角度反驳网页观点，找逻辑漏洞 |

### 暗色模式

强制网页暗色显示，基于 CSS filter 反转实现。优先于页面加载注入，无白屏闪烁。切换即时生效，无需手动刷新。

### 沉浸式翻译

双语对照翻译，捕捉网页中所有英文段落，批量发送至 AI 翻译，译文以透明样式内联展示，不破坏原始排版。

### 阅读模式

基于 Mozilla Readability 提取正文，去除广告和干扰元素，以干净排版呈现文章内容。支持一键进入 / 退出。

### 用户脚本引擎（类 Tampermonkey）

内置用户脚本管理器，兼容 Greasy Fork 等平台的大部分脚本。

**支持的 GM API：**

GM_addStyle、GM_getValue、GM_setValue、GM_deleteValue、GM_listValues、GM_xmlhttpRequest、GM_notification、GM_openInTab、GM_setClipboard、GM_registerMenuCommand、GM_info、GM_log、unsafeWindow

**功能特性：**

- 解析 UserScript 元数据
- URL 匹配引擎（@match、@include、@exclude）
- 三种注入时机（document-start、document-end、document-idle）
- 脚本启用 / 禁用 / 删除
- 一键安装（点击 .user.js 链接自动弹出安装对话框）
- 脚本持久化存储
- GM_xmlhttpRequest 跨域请求（通过主进程 net.fetch 绕过 CORS）

**兼容覆盖率：** 约 85-90% 的热门用户脚本

### 资源嗅探

自动检测网页中的媒体资源（视频、音频、图片、文档），在地址栏显示资源数量角标，点击展开资源列表，支持复制链接和下载。

### 外部下载器对接

支持将下载任务发送至 IDM、FDM、NDM 等外部下载工具。自动检测已安装的下载器，可在设置中选择默认下载器和切换拦截模式。

### 标签页预览

鼠标悬停标签页时显示页面缩略图预览，400ms 延迟触发，自动缓存。

### 侧边栏网页面板

在浏览器左侧打开一个独立的网页面板，可固定常用网页（ChatGPT、翻译、微信等），支持自定义 URL 和面板保存。

### 鼠标手势

按住右键拖动触发快捷操作：

| 手势 | 动作 |
|------|------|
| ← | 后退 |
| → | 前进 |
| ↑ | 新建标签页 |
| ↓ | 关闭当前标签页 |
| ↓→ | 刷新 |
| ↑↓ | 强制刷新 |
| ↓↑ | 恢复关闭的标签页 |
| ←→ | 滚动到页面顶部 |
| →↓ | 滚动到页面底部 |

手势轨迹实时可视化显示。

### 分屏浏览

支持左右分屏同时浏览两个网页，可从右键菜单将链接打开到右侧，支持左右互换。

### 广告拦截

内置广告拦截功能，支持白名单管理。

## 视觉风格

Zhi Browser 2.0.0 采用精修后的深色桌面应用界面，参考 Arc Browser 和 Zen Browser 的质感方向，但保持自己的横向标签页形态。

主要视觉特性：

- 统一的深色主题色系
- 主题色可配置
- 字体可切换
- 精修标签栏、地址栏、侧边面板
- 工具栏按钮统一风格
- 过渡动效和交互反馈

## 打包说明

本地正式打包命令：

```
npx electron-builder --config electron-builder.yml --win nsis --x64 --publish never
```
